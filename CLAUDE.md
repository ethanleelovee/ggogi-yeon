# CLAUDE.md — 작업 규칙

## 1. 수정사항 범위
- 요청한 부분만 수정한다. 나머지 코드는 반드시 그대로 유지한다.
- "다른 건 유지"라는 말이 없어도 이 규칙은 항상 적용된다.

## 2. 수정이 불가능한 경우
- 요청한 수정이 다른 부분에 영향을 줄 수밖에 없다면, 실행 전에 반드시 알린다.
- 선택지 2가지를 제시하고 확인을 받은 뒤 진행한다.

## 3. 명령 이해 및 확인
- 애매하거나 정보가 부족한 부분이 있으면 실행 전에 질문한다.
- 이해한 내용을 먼저 설명하고, 확인을 받은 뒤 진행한다.
- 헷갈리는 게 없으면 바로 진행해도 된다.
